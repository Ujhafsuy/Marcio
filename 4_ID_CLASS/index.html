ID e Eclass

ID

O id serve como um CPF. Assim como só posso ter um número de CPF cada elemento só pode ter um valor de ID. O CPF é único, ou seja, ninguém pode ter um CPF igual ao meu. Isso significa que o ID também é único, então não é possivel repetir o mesmo valor duas vezes.

Class

O Class funciona de maneira oposta ao ID. Diferente do ID em que você só oide ter um valor, no Class você pode ter vários valores. Também diferente do ID, em que não se pode repetir um valor, no Class você pode utilizar o mesmo valor para vários elementos.
